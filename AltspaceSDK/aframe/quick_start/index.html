altspace developer quick start

Important!

Use of a local webserver and basic HTML knowledge is required for this quick start.
A basic local webserver is outside the scope of this tutorial, but here is a good place to start:

Simple Web Server for Mac:
https://lifehacker.com/start-a-simple-web-server-from-any-directory-on-your-ma-496425450
Web Server for Windows 10:
https://www.microsoft.com/en-us/download/details.aspx?id=48264

If you already know how to download,install and run a local web development server, continue.
Developer Registration is recommended.

TL;DR

This tutorial will show you how to:
  1. Prepare your "AltspaceVR Homespace" for full space world development
  2. Get the sample code up and running.

1. Your "homespace" Settings

Login to the AltSpaceVR website https://account.altvr.com/and go to "Settings".

2. Click the "Edit Button".

3. Set your "Environment" to "SDK Playground Homespace",and click "Update" to save your configuration.

4. Create your local "index.html" file

Sample Code
<html>
  <head>
    <title>My A-Frame Scene</title>
    <script src="https://aframe.io/releases/0.7.0/aframe.min.js"></script>
    <script src="https://cdn.rawgit.com/AltspaceVR/AltspaceSDK/v2.9.0/dist/altspace.min.js"></script>
  </head>
  <body>
    <a-scene altspace>
      <a-entity geometry="primitive: box" material="color: #C03546"></a-entity>
    </a-scene>
  </body>
</html>

Copy the "Sample Code" to a local HTML file named "index.html" and start your local web server in that directory.

5. Start "AltSpaceVR"

6. Enter "empty" homespace

An "empty and ready to go" development space should await you after login. It should look like this:
7. Open the "URL bar" in front of you and enter the address to your local webserver root (in this example, mine is http://localhost:8080)
